___________
DESCRIPTION
-----------
CryptoGraph is a program designed and implemented to analyse cryptocurrency trading data

_____
USAGE
-----
Compilation:
    javac -cp json-20200518.jar *.java

Execution 
* Interactive 
    java -cp json-20200518.jar:. CryptoGraph -i
* Report 
    java -cp json-20200518.jar:. CryptoGraph -r <asset filename> <trade filename>
